# Cafeteria Sales Analysis Using Python & Pandas

## Project Overview

This project analyzes cafeteria sales transaction data containing 10,000 records collected during the year 2023. The objective is to clean the raw data, perform exploratory data analysis (EDA), identify business insights, and suggest strategies to improve revenue and operational efficiency.

The project demonstrates practical usage of:

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## Problem Statement

Cafeterias generate large volumes of sales data daily, but this data is often underutilized. Without proper analysis, businesses may miss opportunities to:

* Improve profitability
* Identify customer preferences
* Optimize inventory
* Increase revenue
* Reduce waste

This project focuses on analyzing cafeteria sales data to uncover meaningful business insights.

---

## Dataset Information

* Total Transactions: 10,000
* Time Period: January 2023 – December 2023
* Dataset Fields:

  * Transaction Date
  * Item
  * Quantity
  * Price Per Unit
  * Total Spent
  * Payment Method
  * Location

## Data Cleaning Performed

The following preprocessing steps were completed:

* Replaced invalid values such as `UNKNOWN`, `ERROR`, and blank spaces
* Handled missing values
* Converted data types:

  * Date → datetime
  * Quantity → integer
  * Price/Total Spent → numeric
* Removed duplicate records
* Filled missing values using logical mapping techniques
* Sorted transactions by date

---

## Exploratory Data Analysis (EDA)

The following analysis and visualizations were performed:

* Correlation Analysis
* Scatter Plot:

  * Quantity vs Total Spent
  * Price per unit vs Total spent
* Bar Chart:

  * Item-wise Revenue
* Line Chart:

  * Monthly Revenue Trend
* Histogram:

  * Total Spending Distribution
* Box Plot:

  * Outlier Detection

---

## Key Insights

* Monthly revenue remained stable between $6,700 and $7,400
* June and October recorded the highest sales
* February showed the lowest revenue
* Salad, Sandwich, Smoothie, and Juice generated the highest revenue
* Most customers spent between $7 and $12 per order
* Healthy and fresh food items were preferred by customers

---

## Business Recommendations

* Introduce new healthy food variations
* Create combo offers such as:

  * Salad + Smoothie
  * Sandwich + Juice
* Improve promotions for low-performing items
* Maintain inventory for high-demand products
* Offer loyalty coupons to regular customers
